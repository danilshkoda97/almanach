    almanc v1.0

    almanc - программа для хранения информации о проблемах, которые появляются у клиентов с iiko, способах их решений

    Прогрмамма реализована на языке программирования Python с использованием базы данных SQLite. Графический инферфейс БД -
SQLiteStudio, среда разработки - PyCharm. Приложение оконное, БД - локальная

    Главные функции:
    1. Добавление категорий проблем
    2. Добавление названий проблем
    3. Добавление информации о проблеме, если это необходимо
    4. Добавление информации о решении проблемы
    5. Возможность изменять категории, названия, информацию и способы решения проблем
    6. Возможность удалять проблемы и способы решения проблем
    7. Поиск информации и решения проблемы по ключевым словам
    8. Поиск информации и решения проблемы по общему списку

    Структура базы данных:
    1. categories (id, name) - таблица, предназначенная для хранения названий категорий
    2. problems (id, name, categori, information, decision) - таблица, которая хранит в себе всю информацию о проблемах и
способах их решений

    Этапы разработки:
    1. Создание БД, таблиц +
    2. Подключение к БД из приложения +
    3. Создание графического интерфейса
        3.1 Вёрстка блока работы с категориями +
        3.2 Вёрстка блока для работы с проблемами
    4. Работа с категориями +
        4.1 Реализация добавления категории +
        4.2 Проверка "если такая категория уже существует, то вывести соответствующее сообщение" +
        4.3 Реализация изменения категории +
    5. Работа с проблемами +/-
        5.1 Реализация добавления проблем +
        5.2 Проверка "если такая проблема уже существует, то вывести соответствующее сообщение" +
        5.3 Реализация редактирования проблем +/-
            5.3.1 Проверка "если ничего не было изменено, то вывести соответствующее сообщение"
        5.4 Реализация удаления проблем +
        5.5 Поиск и показ существующих проблем +
    6. Удаление регистрозависимости +
    7. Выстроить иерархию файлов внутри проекта
